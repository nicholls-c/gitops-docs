# Local Setup

This guide assumes some local tooling.

- [`Windows Subsystem for Linux 2`](https://learn.microsoft.com/en-us/windows/wsl/install)
- A `Debian` based distro ([`Ubuntu-24.04`](https://documentation.ubuntu.com/wsl/stable/howto/install-ubuntu-wsl2/) at time of writing)
- [`kubectl`](https://pwittrock.github.io/docs/tasks/tools/install-kubectl/)
- [`minikube`](https://minikube.sigs.k8s.io/docs/start/?arch=%2Fwindows%2Fx86-64%2Fstable%2F.exe+download)
